Rowing Classes
Classes that implement the business logic of rowing signups
Auxiliary  
Attendance Defines valid attendance values for a participant in a rowing session
BoatView A read-only snapshot of boat data
OarsetView A read-only snapshot of oarset data
SeatPreference Defines valid seating preferences for a rowing signup
Seat An assignment of participant to a specific position within a boating
Boating  
BoatingView A read-only snapshot of boating data
BoatingState Defines valid states of a boating
IBoating A read-write interface to boating data, used only by RowingSession beans
IBoatingHome A factory that manufactures instances of IBoating, used only by RowingSession beans
remote/BoatingBean A remote implementation of IBoating
remote/BoatingBeanState A utility used to implement remote/BoatingBean
Participant  
ParticipantView A read-only snapshot of participant data
ParticipantState Defines valid states of a participant
IParticipant A read-write interface to participant data, used only by RowingSession and Boating beans
IParticipantHome A factory that manufactures instances of IParticipant, used only by RowingSession and Boating beans
remote/ParticipantBean A remote implementation of IParticipant
remote/ParticipantBeanState A utility used to implement remote/ParticipantBean
Rowing  
RowingSessionView A read-only snapshot of a rowing session
RowingSessionState Defines valid states of a rowing session
IRowingSession A read-write interface to a rowing session
IRowingSessionHome A factory that manufactures instances of IRowingSession
remote/RowingSessionBean A remote implementation of IRowingSession
remote/RowingSessionBeanState A utility used to implement remote/RowingSessionBean
Exception  
RowingException Thrown on a general error related to the rowing package
BoatingStateException Thrown when a boating is in an inconsistent state, or when a boating operation is invoked on an inappropriate state
ParticipantStateException Thrown when a participant is in an inconsistent state, or when a participant operation is invoked on an inappropriate state
RowingSessionStateException Thrown when a rowing session is in an inconsistent state, or when a rowing session operation is invoked on an inappropriate state
Deprecated  
SessionSet deprecated; moved to web package
SessionView deprecated; moved to web package
